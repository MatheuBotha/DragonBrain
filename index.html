Project Description

SwarmVis

SwarmViz is a program that visualises how optimisation algorithms work. Optimisation algorithms are a type of Artificial Intelligence that is used to automatically solve complex real-world problems in science, engineering, finance, and military research & development. With SwarmViz anyone can gain an understanding of how these sophisticated problem-solving algorithms work by letting the user play around with trying different optimisation algorithms to solve different example problems.
